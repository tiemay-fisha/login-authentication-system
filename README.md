# 🔐 Secure Authentication System

## 👋 Welcome

This project is a **Secure Authentication System** built using Python.  
It is designed as a learning project to understand how real-world login systems work in terms of security, validation, and user management.

The goal of this project is not just to build a login system, but to understand the **core ideas behind cybersecurity and authentication systems**.

---

## 🎯 Project Purpose

I built this project to learn and demonstrate:

- How users register and log in securely
- How passwords are protected using hashing
- How systems validate user input
- How login attempts can be tracked

This project is part of my journey into **software engineering and cybersecurity fundamentals**.

---

## ⚙️ Features

✔ User Registration System  
✔ Secure Password Hashing (SHA-256)  
✔ Password Strength Validation  
✔ Password Confirmation Check  
✔ Login System with Retry Handling  
✔ Login History Tracking  
✔ Input Validation and Error Handling  
✔ Simple and clean command-line interface  
✔ User-friendly messages and prompts  

---

## 🔐 Security Concepts Used

This project applies basic but important cybersecurity principles:

- **Hashing (SHA-256)** to store passwords securely
- **Input validation** to reduce incorrect or weak data
- **Authentication logic** to verify users safely
- **Separation of concerns** between registration and login systems

These concepts are widely used in real-world systems such as websites, apps, and banking systems.
